# One · 一个 For .Net
---
## 简介：
- 使用 .NET Core 为基础框架
- Core项目为核心库，功能是爬[One·一个官网](http//www.wufazhuce.com)的网页数据
- 桌面客户端暂包含WPF,UWP，引用的Core项目获取数据
- WebAPI项目路由地址为/oneapi/[Controller]
- Blazor应用采用WebAssembly客户端框架，由于不能跨域访问，未使用Core去获取数据，需要搭配WebAPI服务程序使用
